SWATCH WATCH

DESCRIPTION: My object is a black Swatch watch with a light green circle shape dial, outside of the dial there is a blue ring above it. The blue ring can be move and rotate. There are numbers(regular clock numbers) both inside the dial and outside on the blue ring. I created this watch based on the description given by Selina in class, it was a really fun project to kick start our coding journey in p5.js.
